#### Description:
 > a series rest api

#### Dependencies
> django
> djangorestframework


#### How to run:
make sure your virtual environment is activated
run ```pipenv install -r requirements.txt```
to run server, ```python manage.py runserver```

on your web browser, go to ```http://127.0.0.1:8000/series/ ```
